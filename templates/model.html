Used Deep Learning Models

Model 1: CNN

A Convolutional Neural Network, also known as CNN or ConvNet, is a class of neural networks that specializes in processing data that has a grid-like topology, such as an image. A digital image is a binary representation of visual data. It contains a series of pixels arranged in a grid-like fashion that contains pixel values to denote how bright and what color each pixel should be. CNNs are particularly effective in identifying patterns such as edges, textures, and shapes. They are widely used in computer vision tasks like object detection, image classification etc.

Model 2: EfficientNet

EfficientNet is a type of Convolutional Neural Network (CNN) that is designed to achieve high accuracy while maintaining optimal efficiency. Unlike traditional CNNs that scale depth, width, or resolution independently, EfficientNet uses a compound scaling method to uniformly scale all dimensions in a balanced way. This architecture allows the model to deliver better performance with fewer parameters and less computational cost. It is well-suited for image classification tasks, offering improved accuracy and faster inference on complex datasets.
